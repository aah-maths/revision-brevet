# Sections de solides et démonstrations

## Objectif

Reconnaître une section plane et rédiger une démonstration géométrique complète.

## Sections de solides

Une section est la figure obtenue quand un plan coupe un solide.

- cube ou pavé coupé parallèlement à une face : rectangle de mêmes dimensions que
  cette face ;
- cylindre coupé parallèlement à sa base : disque de même rayon ;
- cylindre coupé perpendiculairement à sa base en passant par l’axe : rectangle ;
- cône coupé parallèlement à sa base : disque ;
- sphère coupée par un plan : disque ; son bord est un cercle.

Dans une perspective cavalière, les longueurs en profondeur ne sont généralement pas
représentées à leur vraie mesure. Les arêtes cachées sont dessinées en pointillés.

## Rédiger une démonstration

Une démonstration suit trois étapes :

1. données : ce que l’énoncé fournit ;
2. propriété : le théorème utilisé ;
3. conclusion : ce que la propriété permet d’affirmer.

### Modèle avec Pythagore

Dans ABC, le plus grand côté est $BC$.  
$AB^2+AC^2=...$ et $BC^2=...$.  
Les deux résultats sont égaux. D’après la réciproque du théorème de Pythagore,
le triangle ABC est rectangle en A.

### Modèle avec Thalès

Il faut citer les alignements, établir l’égalité des rapports puis utiliser la
réciproque pour conclure que les droites sont parallèles.

## Erreurs fréquentes

- conclure uniquement à partir d’un dessin ;
- écrire « d’après Pythagore » sans vérifier que le triangle est rectangle ;
- oublier les conditions d’alignement dans la réciproque de Thalès ;
- confondre la section, qui est plane, avec le solide entier.
