# Fonctions

## Objectif

Lire, calculer et représenter une relation entre deux grandeurs.

## Vocabulaire

Si $f(3)=7$, alors 7 est l’image de 3 et 3 est un antécédent de 7.

Pour $f(x)=ax+b$, la représentation est une droite. $a$ est le coefficient
directeur et $b=f(0)$ est l’ordonnée à l’origine.

- fonction linéaire : $f(x)=ax$, situation de proportionnalité ;
- fonction affine : $f(x)=ax+b$.

Sur un graphique, trouver l’image consiste à partir de l’axe horizontal. Trouver un
antécédent consiste à partir de l’axe vertical.

## Erreurs fréquentes

- confondre image et antécédent ;
- oublier les parenthèses avec un nombre négatif ;
- croire que toute courbe représente une proportionnalité.
