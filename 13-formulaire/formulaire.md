# Formulaire final du brevet

## Calcul

- Priorités : parenthèses, puissances, produits/quotients, sommes/différences.
- $a^m a^n=a^{m+n}$ ; $a^m/a^n=a^{m-n}$.
- Hausse de $t\%$ : multiplier par $1+t/100$.
- Baisse de $t\%$ : multiplier par $1-t/100$.
- $v=d/t$, $d=v\times t$, $t=d/v$.

## Calcul littéral

- $k(a+b)=ka+kb$.
- $(a+b)^2=a^2+2ab+b^2$.
- $(a-b)^2=a^2-2ab+b^2$.
- $(a-b)(a+b)=a^2-b^2$.

## Données

- moyenne $=\text{somme}/\text{effectif}$ ;
- étendue $=\text{maximum}-\text{minimum}$ ;
- probabilité $=\text{cas favorables}/\text{cas possibles}$ si équiprobabilité ;
- $P(\overline A)=1-P(A)$.

## Géométrie

- Pythagore : hypoténuse² = côté² + côté².
- Thalès : rapports de longueurs correspondantes égaux si les droites sont parallèles.
- $\cos=\text{adjacent}/\text{hypoténuse}$.
- $\sin=\text{opposé}/\text{hypoténuse}$.
- $\tan=\text{opposé}/\text{adjacent}$.
- Somme des angles d’un triangle : 180°.

## Aires et volumes

- triangle : $bh/2$ ;
- disque : $\pi r^2$ ;
- cylindre : $\pi r^2h$ ;
- pyramide ou cône : $A_{\text{base}}h/3$ ;
- boule : $4\pi r^3/3$.
